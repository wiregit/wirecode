Here are the important tests:

-UltrapeerTester: tests ultrapeer functionality and basic message
 routing (OOP/SINT)
-LeafTester: tests leaf node functionality (OOP/INT)
-AcceptLimitTest: checks connections lots (IP/AUTO)
-downloader/DownloadTester: checks swarm downloading (IP/AUTO)
-UploadTester: checks uploading (OOP/AUTO)
-SearchViewTest: checks GUI for displaying search results (IP/AUTO)

Key:
    OOP=out of process, can be used for other vendors
    IP=in process

    AUTO=automatic
    INT=interactive
    SINT=semi-interactive
